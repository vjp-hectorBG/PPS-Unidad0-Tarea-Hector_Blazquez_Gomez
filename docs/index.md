---
## Descripción
El proyecto que tenemos entre manos trata sobre la creación de un repositorio documentado que implementa un flujo de trabajo automatizado para generar y publicar documentación técnica. Utilizo **MkDocs** para crear una web estática con la documentación, que posteriormente despliego en un contenedor NGINX mediante Docker, demostrando así la integración de múltiples tecnologías en un ciclo de desarrollo seguro.
