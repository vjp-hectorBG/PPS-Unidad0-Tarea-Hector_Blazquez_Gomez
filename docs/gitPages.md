# Git Pages
---
Aquí explico brevemente como activar Github Pages.

---

Para empezar nos deberemos situar en la página de nuestro poyecto en Github y fijarnos en la barra superior derecha la cual tiene una opción que pone **Settings** , deberemos acceder a ella y en la barra izquierda deberá de estár una opcion que pone **Pages**

![Settings](https://raw.githubusercontent.com/vjp-hectorBG/PPS-Unidad0-Tarea-Hector_Blazquez_Gomez/refs/heads/main/imgs/settings.png)

![Pages](https://raw.githubusercontent.com/vjp-hectorBG/PPS-Unidad0-Tarea-Hector_Blazquez_Gomez/refs/heads/main/imgs/pages.png)

Después veremos un apartado que pone **Branch** , ahí elegimos gh-pages y le damos al botón **save**.

![save](https://raw.githubusercontent.com/vjp-hectorBG/PPS-Unidad0-Tarea-Hector_Blazquez_Gomez/refs/heads/main/imgs/save.png)
