# Git
---

En este documento especifico como he creado mi repostorio de Github paso a paso.

---
## -1 
Desde la página oficial de [Github](https://github.com/) he clickado en el botón superior izquierdo en el que pone **"NEW"** , este te llevará a una ventana emergente , en esta especifiqué el título del repositorio (En mi caso **PPS-Unidad0ActividadGIt-HectorBG**).

![crear repositorio]()

Es muy importante que aquí le haya dado a "Crear README.md" si no Github crea un repositorio vacío y hubiese tenido que hacer más pasos.

## -2 
Una vez creado el repositorio , en mi máquina Kali he creado una carpeta para mis proyectos y ejecuté en la terminal `git clone https://github.com/vjp-hectorBG/PPS-Unidad0-Tarea-Hector_Blazquez_Gomez.git` , este comando me descargó mi repositorio nuevo a mi máquina y pude empezar a trabajar.
